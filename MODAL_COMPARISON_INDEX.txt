FIELD DIFFERENCES
=================
Quotation IDs:          Order IDs:
- quotationDate         - orderDate
- deliveryDate          - orderDeliveryDate
- selectedCustomerCode  - selectedSupplierCode
- selectedCustomerName  - selectedSupplierName
- quotationTitle        - orderTitle
- quotationRemark       - orderRemarks
- quotationDetailTable  - orderDetailTable
- totalSupplyPrice      - orderTotalSupplyPrice
- totalVat              - orderTotalVat
- grandTotal            - orderGrandTotal

ORDER ONLY (Quotation does NOT have):
- orderPaymentMethod dropdown
- orderStatus dropdown


CRITICAL DIFFERENCES
====================
1. Extra Fields in Order Modal
   - Payment Method dropdown (between supplier selection and title)
   - Status dropdown (same row as payment method)

2. ID Naming Inconsistency
   - Order table body: "newOrderDetailTableBody" (inconsistent "new" prefix)

3. Submit Button Text
   - Quotation: "견적서 등록"
   - Order: "저장"

4. Search Function Names
   - openMaterialSearchModal() vs openOrderMaterialSearchModal()
