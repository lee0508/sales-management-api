SQL Agent 작업 등록 (정기 실행)
이제 이 스크립트를 정기적으로 실행하도록 설정하는 것을 권장합니다.
SQL Agent 작업 생성 방법

SSMS에서 SQL Server 에이전트 → 작업 → 우클릭 → 새 작업
일반 탭:

이름: YmhDB 인덱스 유지보수
설명: 매주 일요일 새벽 인덱스 재구성 작업


단계 탭 → 새로 만들기:

단계 이름: 인덱스 유지보수 실행
유형: Transact-SQL 스크립트(T-SQL)
데이터베이스: YmhDB
명령: 위의 전체 스크립트 복사


일정 탭 → 새로 만들기:

이름: 매주 일요일 새벽 2시
일정 유형: 되풀이
빈도: 매주 일요일
시간: 새벽 02:00


확인 클릭

권장 실행 주기

소규모 DB (10GB 이하): 매주 1회
중규모 DB (10-50GB): 매주 1회
대규모 DB (50GB 이상): 2주에 1회
